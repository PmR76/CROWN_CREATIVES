What you need to provide for each new project

1. still.jpg
A static preview image.
If you don’t provide one, the engine will auto‑generate a YouTube thumbnail.

2. motion.mp4
A short 2–4 second looping clip.
If you don’t have one, you can reuse the still image.

3. config.json
Just fill in the template.

4. Folder name
